BitLut Privacy Policy Last updated: May 2026 BitLut is an Android utility application designed to synchronize fitness and health activity data between Huawei Health and Android Health Connect after explicit user authorization. Data Processing BitLut may access the following categories of data after user consent: * step count; * heart rate data; * synchronization timestamps; * technical diagnostic logs generated locally on the device. Data Usage The application uses this data only for: * synchronization between Huawei Health and Android Health Connect; * displaying synchronization status; * troubleshooting synchronization problems. No Sale of Data BitLut does not sell personal data. No Advertising SDKs BitLut does not use advertising SDKs. Local Processing Health data is processed locally on the user's device whenever possible. Third-Party Services BitLut interacts with: * Huawei Health / HMS Health Kit; * Android Health Connect. User Control Users may revoke permissions or uninstall the application at any time. Medical Disclaimer BitLut is not a medical application and does not provide diagnosis, treatment, or medical advice. Contact GitHub Repository: https://github.com/spntext-jpg/BitLut
